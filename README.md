# Sistema de Contabilidade — Front-end

Front-end do sistema de contabilidade desenvolvido para integração com API back-end. O projeto contempla a estrutura visual, componentes e fluxo de navegação da aplicação.

---

## Objetivo do Projeto

Fornecer a interface do usuário do sistema contábil, com layout organizado e pronto para integração com serviços back-end, facilitando o trabalho em equipe entre front-end e back-end.

---

## Escopo do Front-end

Este repositório é responsável por:

* Estrutura das telas
* Layout e responsividade
* Organização de componentes


---

## Tecnologias Utilizadas

* HTML
* CSS
* JavaScript

---


## Status do Projeto

**Em desenvolvimento** — estrutura front-end pronta para evolução e integração.
